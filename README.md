YMTLib
====================

## 概要

やまと(waco)製作のMODで使用している共通ライブラリです。


## 前提

ModLoader が必要です。
MinecraftForge でも動くかもしれません。


## 導入

mods に zip のまま放り込んでください。
導入前にバックアップを取るのも忘れずに。".minecraft" 自体をバックアップするのがお手軽です。
